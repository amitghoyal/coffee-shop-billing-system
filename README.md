# Coffee Shop Billing System

A full-stack Coffee Shop Billing System built with:
- **Backend**: Java (OOP concepts, billing logic, order management)
- **Frontend**: React (user interface for menu, order form, billing)
- **Database**: MySQL (menu items, orders, payments)

## Features
- Menu management
- Order placement
- Bill generation
- Payment handling
